# ProximityVC
Proximity Voice Chat with Socket.IO

### Installation
1. Install [Node.js](https://nodejs.org/en/)
2. `cd ProximityVC`
3. `npm i`
4. `node server.js` (use [forever](https://www.npmjs.com/package/forever))
